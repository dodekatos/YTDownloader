Having issues with Windows Defender flagging the Native Client or abruptly stopping FFmpeg re-encodes?

Open your Antivirus (or click the button below), and navigate through:
Add an exclusion >> Folder >> C:\ProgramData\YTDownloader\bin

You will see an Administrator prompt (UAC) after clicking the button.
This is normal Windows behaviour, because you're opening the Windows Defender Exclusions page. For the sake of transparency, the addon will not add the exception automatically. You can add the exception yourself.
